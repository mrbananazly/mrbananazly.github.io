搭建及部署

参考博文链接：Win10下Hexo博客搭建教程，及阿里云服务器部署实战 (qq.com)

博文发布

在本地机器上部署Hexo相关MyHexoBlogs文件夹下，进入MyHexoBlogs/myblogs/source/_posts目录；

在当前页进入git bash，输入hexo clean 清除缓存，hexo g 解析静态文件，hexo d 刷新部署新的资源。

hexo cl && hexo g && hexo d
